## 二、首页（所有人可见）

```
导航栏: 首页 → 服务大厅 → 优惠券 → 服务商中心 → 加盟 → 登录/注册
```

- **覆盖全行业** — 医疗/美容/健身/教育/家政/法律 6 个分类，点击跳转对应筛选
- **优惠服务** — 展示有优惠券的服务商及其折扣信息
- **猜你喜欢** — 个性化推荐（根据历史预约分类 + 热度 + 评分排序）
- **热门服务** — 全平台最热门的服务排行

## 三、普通用户

### 注册/登录

```
首页 → 登录 → 注册新账号 → 填写信息 → 注册成功
```

### 我的预约

```
登录后 → 导航栏"我的预约"
```

- 查看所有预约及状态：**待确认 / 预约成功 / 已完成 / 已取消**
- **支付状态**：未支付 / 已支付 / 已退款
- 状态为**未支付**时 → 点"**去支付**" → 弹窗 → 点"**前往支付宝支付**" → 完成支付*//这个功能没修好*
- 待确认/预约成功状态 → 点"**取消预约**"
- 已完成 → 点"**评价**"（打分 + 评论）

### 预约服务流程

```
首页 → 浏览服务/热门推荐 → 点进服务详情 → 选日期 → 选时间 → 填写备注 → 确认预约
→ 预约成功 → 在我的预约查看 → 去支付 → 等待服务商确认
```

### 优惠券

```
导航栏 → 优惠券 → 查看已领取/可领取的优惠券
```

### 个人中心

```
导航栏 → 个人中心 → 修改资料
```

## 四、服务商

### 登录

```
使用 zhangsan / 123456 登录
或 注册普通用户 → 提交服务商入驻申请 → 管理员审核通过 → 成为服务商
```

### 服务商入驻（审核流程）

```
普通用户 → 导航栏"服务商中心" → 提交入驻申请（名称/分类/简介/地址/电话）
→ 管理员审核 → 审核通过后自动成为服务商
```

### 服务商后台

```
登录后 → 导航栏"服务商中心" → 进入服务商后台
```

#### ① 我的服务

- 发布新服务（名称/分类/价格/时长/简介）
- 编辑/删除已有服务

#### ② 预约管理

- 查看所有预约（客户名/服务/日期/时间）
- **确认** → 预约状态变为"已确认"，客户看到"预约成功"
- **完成** → 服务已完成，计入收入统计
- **取消** → 预约取消

#### ③ 优惠券管理

- **创建优惠券**（名称/类型：满减/折扣/数量/有效期）
- 编辑/删除优惠券
- 优惠券自动展示在客户端的"优惠服务"模块

#### ④ 数据统计

- **摘要卡片**：总预约数 / 总收入 / 平均评分 / 待处理
- **收入趋势**：按天/月/年查看柱状图
- **预约状态**：环形图展示各状态占比
- **服务统计**：每个服务的预约数/收入/评分

#### ⑤ 服务商信息

- 修改名称/分类/简介/地址/电话
- **营业时段设置**：周一到周日，每天可设营业时间（如 09:00-18:00）
  - 客户预约时会根据营业时段显示可选时间

