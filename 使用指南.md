### 浏览与搜索服务

```
导航栏"服务大厅" → 可按分类/价格区间/时长筛选 → 按价格/评分/时长排序
```

```
首页点击服务卡片 → 进入服务详情页
→ 查看服务商信息、服务介绍、价格、时长、用户评价
```

### 预约服务流程

```
服务详情页 → 选日期（根据营业时段） → 选时间（显示可用时段） → 填写备注
→ 点"确认预约" → 预约成功 → 跳转到"我的预约"
```

### 我的预约

```
登录后 → 导航栏"我的预约"
```

**状态说明：**

| 预约状态 | 说明 |
|----------|------|
| 待确认 | 刚预约，等待服务商确认 |
| 预约成功 | 服务商已确认（客户看到"预约成功"，服务商看到"已确认"） |
| 已完成 | 服务商标记服务完成 |
| 已取消 | 客户或服务商取消 |

**支付状态：**

| 支付状态 | 说明 |
|----------|------|
| 未支付 | 刚预约，等待支付 |
| 已支付 | 支付完成 |
| 已退款 | 已退款 |

**操作按钮：**

- **去支付**（未支付时）→ 弹出新窗口 → 自动跳转支付宝沙箱登录页
  → 用沙箱买家账号登录 → 支付完成 → 自动跳回，状态变为"已支付"
- **取消预约**（待确认/预约成功时）→ 预约取消
- **评价**（已完成时）→ 打分 + 写评论

### 优惠券


## 五、服务商

### 成为服务商

**方式一：使用已注册服务商账号**
```
登录 zhangsan / 123456
```

**方式二：提交入驻申请**
```
注册普通用户 → 导航栏"服务商中心" → "成为服务商"
→ 填写名称/分类/简介/地址/电话 → 提交申请 → 等待管理员审核
→ 审核通过后自动成为服务商
```

### 服务商后台

```
登录服务商账号 → 导航栏"服务商中心" → 进入后台
```

后台包含 5 个选项卡：

#### ① 我的服务

- **发布新服务**：填写名称/分类/价格/时长/简介 → 发布
- **编辑**：修改已有服务信息
- **删除**：删除服务

#### ② 预约管理

查看所有预约，显示：客户名、服务名、日期时间、金额

| 按钮 | 作用 |
|------|------|
| **确认** | 确认预约 → 客户看到"预约成功" |
| **完成** | 服务完成 → 计入收入统计 |
| **取消** | 取消预约 |

#### ③ 优惠券管理

- **创建优惠券**：名称、类型（满减/折扣）、数量、有效期
- 编辑/删除优惠券
- 创建的优惠券自动展示在客户端的"优惠服务"模块

#### ④ 数据统计

- **摘要卡片**：总预约数 / 总收入 / 平均评分 / 待处理数量
- **收入趋势**：按天/月/年查看柱状图
- **预约状态**：环形图展示各状态占比（待确认/已确认/已完成/已取消）
- **服务统计**：每个服务的预约数/收入/评分表格

#### ⑤ 服务商信息

- 修改名称/分类/简介/地址/电话
- **营业时段设置**：周一到周日，每天独立设置营业时间（如 09:00-18:00）
  - 客户预约时根据营业时段自动计算可用时间

## 七、支付说明

当前使用 **支付宝沙箱环境**（测试用，不会真实扣款）。

### 沙箱买家账号

打开 https://open.alipay.com → 控制台 → 沙箱环境
页面下方有**买家账号**（格式 `xxx@sandbox.com`），复制该账号和密码用于登录支付宝支付页面。

### 切换 Mock 模式

如果沙箱不稳定，可切换为 Mock 模式（无需支付宝账号，即时成功）：

编辑 `alipay-gateway/.env`，将 `MOCK_MODE=false` 改为 `MOCK_MODE=true`，重启网关即可。

### 支付流程

```
我的预约 → 点"去支付" → 弹出新窗口
→ 跳转支付宝沙箱登录页 → 用买家账号登录支付
→ 支付完成 → 自动跳回"我的预约" → 状态变为"已支付"
```
